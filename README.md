# 📚 AppUntes

Aplicación móvil de organización académica para estudiantes de educación superior.

## 📱 Descripción

AppUntes centraliza materias, tareas y fechas de entrega en un solo lugar, reemplazando la dispersión de información entre WhatsApp, cuadernos y aulas virtuales.

## ✨ Funcionalidades

- Gestión completa de materias (CRUD)
- Creación y seguimiento de tareas con prioridad
- Adjuntar fotos desde la cámara del dispositivo
- Calendario académico mensual
- Recordatorios automáticos con WorkManager
- Onboarding inicial con DataStore
- Dark mode y Dynamic Color (Material Design 3)

## 🏗️ Arquitectura

- **Patrón:** MVVM + Repository Pattern
- **UI:** Jetpack Compose + Material Design 3
- **Base de datos:** Room (SQLite)
- **Navegación:** Navigation Compose
- **Background:** WorkManager
- **HTTP:** Retrofit2 + Gson
- **Lenguaje:** Kotlin 100%
